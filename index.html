Binary Outcomes: A Review

  • binary outcome: categorical random variable that takes two values
  • with binary outcomes typically testing the probability of the event in a population or subgroup

Examples:

  • Treatment response [Yes / No]
  • Mutation [present / absent]
  • Adverse event [Present / Absent]

Binary Outcomes: A Review

Binary outcome:

\[ Y \in \{0, 1\} \]

and for a given predictor profile \(X\):

\[ P(Y = 1 \mid X) = p(X) \]

Interpretation:

  • each patient in \(X\) has an underlying probability of response
  • predictors shift that probability
  • logistic regression is a model for estimating \(p(X)\) in categorical data analysis

Review: Contingency Tables

  • contingency table : structured summary of counts for two categorical variables

Contingency tables have two key roles in categorical data analysis:

  1. Descriptive: summarizes the joint distribution of two categorical variables
  2. Probabilistic: provides empirical estimates of probabilities via proportions

Review: Contingency Tables

  • contingency table : structured summary of counts for two categorical variables

  • provides empirical estimates of probability distributions

  • represents observed realization of multinomial random variable

  • helps form basis for probability modeling

Review: Contingency Tables

A contingency table estimates empirical probabilities:

\[ \hat{P}(Y=i \mid X=j) = \frac{O_{ij}}{\sum_i O_{ij}} \]

Where:

  • \(O_{ij}\): observed count in category \(i\) and group \(j\)
  • Estimates conditional probability directly from observed data

Probabilities describe how biological outcomes differ across groups

Contingency tables and probability estimation: example

Question and Background:

You’re conducting a study that compares tumor response between two treatment groups and you want to determine:

Does the new drug increase tumor response?

Variables of interest:

  • Control: standard therapy
  • Drug: new investigational therapy
  • Outcome: Tumor response at 12 weeks (Yes / No)

Contingency tables and probability estimation: example

Research Question:

Does the new drug increase the probability of response?

tab <- table(dat$drug, dat$response)

  Control 0.6206897 0.3793103

Interpretation:

  • each row is an empirical estimate of the conditional distribution \(P(\text{Response} \mid \text{Treatment [Yes+No]})\)
  • estimates \(P(\text{Yes} \mid \text{Control})\) and \(P(\text{Yes} \mid \text{Drug})\)
  • about 37.9% of patients in the Control group responded to treatment
  • about 43.2% of patients in the Drug group responded

Contingency table limitations and logistic regression as an alternative

Limitation: continuous predictors do not fit naturally in tables

  • even thought the outcome is categorical [binary] many research studies include continuous predictors [biomarkers, lab values, scores]

When using a table:

  • continuous features would need to be binned

  • bin-choice sensitivity could lead to infomration loss

  • logistic regression allows one to keep continuous predictors and models how response probability changes across its range

Limitation: Tables can compare groups one factor at a time

Biomedical inference often needs adjustment:

  • baseline risk differs by features included in the model or table [i.e.: biomarker, age, stage]

Logistic regression estimates treatment effect while accounting for other predictors

Limitation: tables estimate group-specific probabilities

  • 2×2 table gives separate estimates per group

Logistic regression provides a probability model that can:

  • estimate effects (odds ratios) for each feature included within the model
  • support formal model comparison (nested models)

Two-way contingency table

Predictor: Treatment
Outcome: Response

Two-way tables are used to estimate conditional probabilities:

\[ \hat{P}(\text{Response} = \text{Yes} \mid \text{Treatment} = \text{Control}) \]

\[ \hat{P}(\text{Response} = \text{Yes} \mid \text{Treatment} = \text{Drug}) \]

Logistic regression models these probabilities

Logistic regression models conditional probability:

\[ P(Y=1 \mid X) \]

Using log odds:

\[ \log\left(\frac{P(Y=1)}{1 - P(Y=1)}\right) = X\beta \]

Logistic regression estimation of probability of drug response

Contingency table probability of response given treatment:

Control: \(P(Yes∣Control)=0.379\)

Drug: \(P(Yes∣Drug)=0.432\)

Logistic regression generates odds ratio: 1.25

Logistic regression estimation of probability of drug response

Contingency table probabilities can be converted into odds

Probability: \(p=P(Y=1)\)

Odds: \(odds=\frac{p}{1-p}\)

Control probability: \(p=0.379\)

Control odds: \(\frac{0.39}{1-0.379}=0.611\)

Drug probability: \(p=0.432\)

Drug odds: \(\frac{0.432}{1-0.432}=0.761\)

Odds ratio : \(\frac{0.761}{0.611} = 1.25\)

Interpretation:

  • logistic regression shows odds ratios and when contingency table values are converted to odds, the ratio of the odds is the same as the coefficient estimate generated by logistic regression
  • logistic regression also generates a corresponding standard error and p-value

Logistic regression enables multivariable categorical analysis

  • one may choose to model multiple predictors simultaneously versus a binary outcome

Examples include:

  • treatment

  • biomarker level

  • age

  • disease stage

  • contingency tables cannot easily represent multiple predictors simultaneously

  • logistic regression models outcome probability while accounting for multiple predictors

Summary

One-way table:

  • estimates marginal probability

Two-way table:

  • estimates conditional probability

Chi-square test:

  • tests hypotheses about probabilities

Logistic regression:

  • models probabilities as function of predictors
